# Budget-calculator &middot; ![logo](./public/favicon-32x32.png)

## Install and first run

```bash
npm i
cp .env.tpl .env
npm run up
npm run log
```

Then go to http://localhost:3000

## DevEnv actions

Start

```bash
npm run up
```

Show logs

```bash
npm run log
```

Stop

```bash
npm run down
```

Restart

```bash
npm run restart
```

Run Lint

```bash
npm run lint
```

Run tests

```bash
npm run test
```
